# Foundry Setup
Foundry setup options, incuding lists of modules used, and helpful macros

## Modules

##### 5e Statblock Importer
Copy a statblock and it auto generates a creature for it. Makes setting up non-SRD monsters much quicker
https://github.com/jbhaywood/5e-statblock-importer
##### About Time
Used to help automate timed effects
https://gitlab.com/tposney/about-time

##### Active Auras
Helps automate Aura effects
https://github.com/MrPrimate/Active-Auras (make sure to use this fork of it to get to work correctly in Foudry v10)

##### Advanced Spell Effects
Helps with some spell automation
https://github.com/Vauryx/AdvancedSpellEffects

##### Always HP
Helps with HP management
https://github.com/ironmonk88/always-hp

##### Automated Animations
Helps with automating animations 
https://github.com/otigon/automated-jb2a-animations

##### Automated Evocations
Helps with automating summoning spells
https://github.com/theripper93/automated-evocations

##### Compendium Folders
Adds folder structure for much easier organization
https://github.com/earlSt1/vtt-compendium-folders

##### Dfreds Convenient Effects
Greatly Helps Active Effect Management
https://github.com/DFreds/dfreds-convenient-effects

##### Dfreds Effects Panel
Better visibility into a selected tokens active effects
https://github.com/DFreds/dfreds-effects-panel

##### Dice Tray
Adds a dice tray below chat for easier spontaneous rolling
https://gitlab.com/asacolips-projects/foundry-mods/foundry-vtt-dice-calculator

##### Drag Ruler
Shows how far a token has moved while dragging it
https://github.com/manuelVo/foundryvtt-drag-ruler

##### Drop Effects on Items
Allows active effects to be dragged and dropped to and from Item Sheets
https://github.com/ElfFriend-DnD/foundryvtt-drop-effects-on-items

##### Dynamic Active Effects
Adds features onto active effects system
https://gitlab.com/tposney/dae

##### Easy Target
Target tokens with alt+click
https://bitbucket.org/Fyorl/easy-target

##### Enhanced Terrain Layer
Adds a Terrain Layer to Foundry that can be used by other modules to calculate difficult terrain.
https://github.com/ironmonk88/enhanced-terrain-layer

##### FXMaster
Provides various particle and filter effects
https://github.com/ghost-fvtt/fxmaster

##### Initiative Double Click
Double click the initiative value in the combat tracker to quickly modify it.
https://github.com/League-of-Foundry-Developers/fvtt-initiative-double-click

##### Item Macro
Allows macros to be stored in items(very useful with MidiQol)
https://github.com/Kekilla0/Item-Macro

##### JB2A Animated Assets(Free and Patron versions)
Adds a large library of animations to use for animations
https://www.patreon.com/JB2A

##### Let Me Roll That For You!
Helps with roll automation and allows GMs to request rolls from players
https://github.com/League-of-Foundry-Developers/fvtt-module-lmrtfy

##### Magic Items
Adds the ability to make an item magical and add spells/feats to the item
https://gitlab.com/riccisi/foundryvtt-magic-items

##### Midi QOL
Automate the whole combat workflow!
https://gitlab.com/tposney/midi-qol

##### Module Compatibility Checker
Check your module list for compatibility with the latest major version of Foundry VTT.
https://github.com/arcanistzed/mcc

##### Monk's Active Tile Triggers
Use Tiles to automate a bunch of things without haveing to write macros
https://github.com/ironmonk88/monks-active-tiles

##### Monk's Enhanced Journal
Adds a lot of functionality to journals
https://github.com/ironmonk88/monks-enhanced-journal

##### Monk's Hotbar Expansion
Adds functionality to hotbar
https://github.com/ironmonk88/hotbar-expansion

##### Monk's Little Details
Adds a bunch of useful features
https://github.com/ironmonk88/monks-little-details

##### Monk's Scene Navigation
Helps with scene organization
https://github.com/ironmonk88/monks-scene-navigation

##### Monk's Sound Enhancements
Helps with sound organization
https://github.com/ironmonk88/monks-sound-enhancements

##### Monk's Token Bar
Add a token bar to show the current player tokens that are available on the current scene
https://github.com/ironmonk88/monks-tokenbar

##### Monk's Wall Enhancement
Adds some extra wall functionality
https://github.com/ironmonk88/monks-wall-enhancement

##### Perfect Vision
Adds additional token vision functionality
https://github.com/dev7355608/perfect-vision

##### Polyglot
Allows you to pretend to use different fantasy languages
https://github.com/League-of-Foundry-Developers/fvtt-module-polyglot

##### PopOut!
Allows you to pop out windows
https://github.com/League-of-Foundry-Developers/fvtt-module-popout

##### Quick Insert
Allows you tp quicklt search and add something to the scene
https://gitlab.com/fvtt-modules-lab/quick-insert

##### Sequencer
Helps do complex animations
https://github.com/fantasycalendar/FoundryVTT-Sequencer

##### Terrain Ruler
This module makes Foundry's rulers respect difficult terrain that was put down using the Enhanced Terrain Layer module
https://github.com/manuelVo/foundryvtt-terrain-ruler

##### Tidy UI - Game Settings
Makes Game Settings menu much more readable
https://github.com/sdenec/tidy-ui_game-settings

##### Tidy5e Sheet
An alternative Character Sheet for Foundry VTT dnd5e aimed at creating a cleaner UI
https://github.com/sdenec/tidy5e-sheet

##### Times Up
Handles active effects expiring
https://gitlab.com/tposney/times-up

##### Token Action Hud
Makes it easy to do token actions, roll saves, skills, attacks, cast spells, etc without opening the character sheet
https://github.com/Drental/fvtt-tokenactionhud

##### Token Attacher
Attach anything to tokens
https://github.com/KayelGee/token-attacher

##### Token Magic FX
Token Magic is a module for Foundry VTT that allows you to add graphic effects to tokens, tiles, drawings and templates. These FX can be animated.
https://github.com/Feu-Secret/Tokenmagic

##### Token Mold
Can do several things when a token is created, such as roll hit points or randomize names
https://github.com/Moerill/token-mold

##### Token Variant Art
Allows for token variant art, either to provide some variety or to add effects
https://github.com/Aedif/TokenVariants

##### Token Z
Manage the Z index of tokens better
https://github.com/theripper93/token-z

##### Warp Gate
Utility for spawning, modifying, and dismissing tokens on the canvas
https://github.com/trioderegion/warpgate
